    </h3>

..


    | **Class type:**  :ref:`AnyObjectPtrVar`
    | **Eval moment:** :ref:`Const <InsideAMS_EvalMoments>`



    An object-pointer to an AnyFloat that is the target of the rotational widget. If the AnyFloat is an AnyMat33, the rotational widget will create a single rotational matrix as output in the Target.
