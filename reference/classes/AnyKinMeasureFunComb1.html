AnyKinMeasureFunComb1¶

Type

Creatable

Parent class

AnyKinMeasureCombBase

Description¶

This class creates new kinematic measures by applying a parameter function to the input measures.

The parameter function must be of class AnyParamFun. Such functions are sufficiently differentiable to be used to specify smooth kinematics; however, they are scalar input functions, so the input to the functions can only be a single component of a kinematic input measure. If an AnyKinMeasureFunComb1 object has multiple input measures, then it must be supplied with one parameter function for each input measure. Each input is converted to one (or more) output entries by the associated parameter function.
