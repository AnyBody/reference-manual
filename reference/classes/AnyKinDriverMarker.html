AnyKinDriverMarker¶

Type

Creatable

Parent class

AnyKinDriver

Description¶

This class is a specialized kinematic driver class that is aimed at defining marker driver equations for MOCAP (motion capture) analysis.

All constraints are by default defined as soft constraints, which is the typical choice for MOCAP analysis that often lead to over-determinate kinematic systems due to (required) redundancy of the marker set. Moreover, all constraint reaction forces are switched off, because a marker driver function is typically just a trajectory function but not a mechnical actuator governing the motion, kinetically.

Kinematic weighting functions, i.e., time-functions (AnyParamFun) which multiplied to the kinemtic measures before it is used as constraints.
