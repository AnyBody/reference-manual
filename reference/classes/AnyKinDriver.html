AnyKinDriver¶

Type

Creatable

Parent class

AnyKinEqDriver

Child classes

AnyKinDriverMarker, AnyKinMotion

Description¶

This kinematic driver class extends the kinematic-driver base-classes with a simple and a general interface for specifying a time-dependent motion for the driven kinematic measures.

The simpler form of the motion specification uses member variables to specify a constant position, constant velocity motion, or constant acceleration motion. DriverPos0, DriverVel0, and DriverAcc0 are the member variables that govern this simple type of motion specification.

Alternatively, or in addition, single-parameter functions (derived from AnyParamFun) can be added to the driver object and these will be interpreted as functions of time and used as kinematic driver functions.

In case of multiple parallel motion specifications in the same AnyKinDriver-object, all contributions are added together and the motion is driven by the resulting driver function.

Caution

The AnyKinDriver class has reactions (Reaction.Type) enabled by default. This means the driver also provides the force/moments necessary to support the motion.

In some we want, for example, muscles to provide the moments and hence we need to disable the reactions.

Examples¶

Applying a simple kinematical driver by using an AnyKinDriver object with member variables:

AnyKinDriver JointDriver = {
  AnyRevoluteJoint &Jnt = .Joint;
  DriverPos0 = {0.0};
  DriverVel0 = {6.28};
  Reactions.Type = {Off}; // The driver does not provide the moments
  CType = {Hard}; // Contraint must be fullfilled (Other options: Soft, ForceDep)
};

Applying a simple kinematical driver by using an AnyKinDriver object with a single parameter function:

AnyKinDriver JointDriver = {
  AnyRevoluteJoint &Jnt = .Joint;
  AnyFunPolynomial DriverFun = {
    PolyCoef = {{0, 5, -0.4}};
  };
  Reactions.Type = {On};
};
