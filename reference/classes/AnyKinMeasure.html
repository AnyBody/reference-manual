AnyKinMeasure¶

Type

Abstract

Parent class

AnyKinMeasureBase

Child classes

AnyKinMeasureCombBase, AnyKinMeasureReal

Description¶

This class is the abstract base class for all real “kinematic measures”. A “kinematic measure” measures some geometrical quantities based on reference, i.e., AnyRefFrame objects. The main difference to AnyKinMeasureBase is that AnyKinMeasure supplies some output, such as the vectors Pos, Vel, and Acc that contain current values of the measured quantity, see description below.

Notice that the measure is a vector quantity also in cases where it has only one component.
