AnyViaPointLigament¶

Type

Creatable

Parent class

AnyLigamentViaPoint

Description¶

This class is deprecated because it has been renamed to AnyLigamentViaPoint in AnyBody version 7.2. In fact, it is a copy of AnyLigamentViaPoint, which has identical functionality and is kept for backwards. compatibility.
