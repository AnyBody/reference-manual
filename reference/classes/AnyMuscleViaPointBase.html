AnyMuscleViaPointBase¶

Type

Abstract

Parent class

AnyMuscle

Child classes

AnyMuscleSheet, AnyMuscleShortestPath, AnyMuscleViaPoint

Description¶

This is the bases classe for specilizations of the general muscle class, AnyMuscle, that connects two or more so-called via points. The via points are specified as reference frames that must be entered into the object as members.
