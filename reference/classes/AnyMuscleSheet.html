AnyMuscleSheet¶

Type

Creatable

Parent class

AnyMuscleViaPointBase

Description¶

This is a spezialization of the AnyMuscle class defining a muscle for a single string of an AnyKinSPSheet measure. The definition requires an AnyKinSPSheet object and an index of the string in the sheet used for the definition of the muscle. Thus, the class has the same function as an AnyKinMeasureOrganizer selecting a coordinate from the AnyKinSPSheet together with an AnyGeneralMuscle.
