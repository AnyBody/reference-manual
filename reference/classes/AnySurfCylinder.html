AnySurfCylinder¶

Type

Creatable

Parent class

AnyParamSurfAnalytical

Child classes

AnySurfCylinderFit

Description¶

This class contains a parameterized cylinder surface. The cylinder endpoint is located at the origin of the reference frame and it stretches a specified length along Z axis.
