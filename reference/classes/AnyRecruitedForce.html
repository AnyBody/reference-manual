AnyRecruitedForce¶

Type

Abstract

Parent class

AnyAppliedForce

Child classes

AnyMuscle

Description¶

This class is the base class for all muscles and other forces recruited (determined) by the muscle recruitment algorithms in the AnyBodyStudy.
