AnyStdJoint¶

Type

Creatable

Parent class

AnyJoint

Child classes

AnyCylindricalJoint, AnySphericalJoint, AnyUniversalJoint

Description¶

This is the base class for the standard joints, such as spherical and revolute joints. Common for all these standard joints is that the relative motion and the constraints can be defined by a linear and a rotational measure, i.e., AnyKinLinear and AnyKinRotational, respectively. Apart from being a base class, this class also functions as a rigid joint, i.e., fully constrained rigid-body motion.

Examples¶

Here is a simple use-case to constrain all six degrees of freedom between two objects:

AnyStdJoint jnt = {
    AnyRefFrame& obj1 = .Object1;
    AnyRefFrame& obj2 = .Object2;
};

  .WeightFun = ... ¶ Kinematic weighting functions, i.e., time-functions (AnyParamFun) which...

Class type: AnyObjectPtrArray
Eval moment: Const
Default: Depends on other values

Kinematic weighting functions, i.e., time-functions (AnyParamFun) which multiplied to the kinemtic measures before it is used as constraints. An array of vector functions can be entered; in case of more than one function, their outputs are stacked into one vector. The total number of elements must match the dimension of the kinematic measure.
