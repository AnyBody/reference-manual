AnyOptKinStudy¶

Type

Creatable

Parent class

AnyOptStudy

Description¶

This study class is specialized towards optimization of geometrical design parameters according to a kinematic performance measure. The target is to make the kinematic analysis of an over-determinate kinematic system, please refer AnyMechOprKinAnl, perform as good as possible, i.e., with as little kinematical constraint violation as possible over the whole anaysis period.

This type of kinematic optimization is predefined in this class, where the objective design measure is a predefined member in contrast to the general optimization study base-class. In addition, the Analysis can contain only one operation and it must be a kinematic analysis operation, i.e., AnyMechOprKinAnl.

The kinematic paramter optimization can be carried out by executing the base-class Optimization operation. However, this is not recommended because the class is equipped with a specialized operation called ParameterOptimization. This specialized operation is designed to exploit the structure of the optimization problem, which indeed contains an additional optimization inside the kinematical analysis.

The standard Optimization operation together with the kinematical analysis will work as a two-level optimization as could be expected: the inner loop (the kinematical analysis) optimizing positions for each time step and outer loop optimizing geometrical the design variables. The ParameterOptimization operation merges the two optimization problems into one large scale optimization, having both geometrical design variable as well as all positions for all time-steps as unknowns. Efficient exploitation of the structure of this large scale problem allows solving the assembled problem far more efficient than the two-level optimization.
