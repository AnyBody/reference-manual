AnyKinMeasureNormComb¶

Type

Creatable

Parent class

AnyKinMeasureCombBase

Description¶

The norm combination measure contained in this class produce a single measure by calculating a norm-like quantity of the inputs. In addition to a traditional norm, this expression introduce a weight factor for each input as well as an offset value. The output value is defined as by

\[OUTPUT = \left( \sum_i (w_i*INPUT_i + Offset)^p \right)^{1/p} - Offset\]

where i is the index of the inputs and the sum contains all inputs of the measure.
