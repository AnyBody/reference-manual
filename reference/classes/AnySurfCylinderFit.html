AnySurfCylinderFit¶

Type

Creatable

Parent class

AnySurfCylinder

Description¶

This class represents a parameterized cylinder surface fitted to a point cloud. The fitting method computes/defines origin, radii, and orientation of the cylinder based on the input parameters.
