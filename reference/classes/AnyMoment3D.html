AnyMoment3D¶

Type

Creatable

Parent class

AnyAppliedForce

Description¶

A three-dimensional moment applied to a reference frame. The moment can be specified in both global and local coordinates but not simulataneously.

 

Expected members¶

AnyRefFrame (1) ¶ The reference frame to which the moment is applied.

Class type: AnyRefFrame
Number: 1

The reference frame to which the moment is applied.

Optional initialization members¶

M = {0.0, 0.0, 0.0} ¶ Moment vector in global coordinates (not computed when a local moment is specified).

Class type: AnyVec3
Eval moment: VelVar
Default: {0.0, 0.0, 0.0}

Moment vector in global coordinates (not computed when a local moment is specified).

Mlocal = {0.0, 0.0, 0.0} ¶ Moment vector in local coordinates (not computed when a global moment is specified).

Class type: AnyVec3
Eval moment: VelVar
Default: {0.0, 0.0, 0.0}

Moment vector in local coordinates (not computed when a global moment is specified).
