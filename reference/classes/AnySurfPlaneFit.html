AnySurfPlaneFit¶

Type

Creatable

Parent class

AnySurfEllipsoid

Description¶

This class represents a parameterized ellipsoid surface fitted to a point cloud. The fitting method computes/defines origin, radii, and orientation of the ellipsoid based on inpur parameters.
