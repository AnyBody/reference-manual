AnyViaPointMuscle¶

Type

Creatable

Parent class

AnyMuscleViaPoint

Description¶

This class is deprecated because it has been renamed to AnyMuscleViaPoint in AnyBody version 7.2. In fact, it is a copy of AnyMuscleViaPoint, which has identical functionality and is kept for backwards. compatibility.
