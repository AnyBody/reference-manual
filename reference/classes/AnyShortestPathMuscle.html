AnyShortestPathMuscle¶

Type

Creatable

Parent class

AnyMuscleShortestPath

Description¶

This class is deprecated because it has been renamed to AnyMuscleShortestPath in AnyBody version 7.2. In fact, it is a copy of AnyMuscleShortestPath, which has identical functionality and is kept for backwards. compatibility.

SPLine.InitWrapPosVectors = ... ¶ An array of 3-D vectors that specify the initial shape of the string. The...

Class type: AnyMatrix
Eval moment: Const
Default: Depends on other values

An array of 3-D vectors that specify the initial shape of the string. The vectors are ordered as rows of a matrix. For each line segment two vectors must be specified. They are defining two points relative to the line segment’s origin and insertion nodes and the curve’s initial shape will be three straight lines; from the origin to the first point, then to the second, and finally to the insertion point.

SPLine.InitWrapPosVecArr = ... ¶ Object pointer array that must collect pointers to the initial position...

Class type: AnyObjectPtr
Eval moment: Const
Default: Depends on other values

Object pointer array that must collect pointers to the initial position vectors. The initial position vectors are one array of 3D vector (AnyMatrix) for each line segment of the SPLine. Pointer value of ‘None’ is allowed, if a straight line between origin and insertion.

SPLine.SolverType = SPSolverModes ¶ The SolverType defines one of four solver for the wrapping problem. Possible...

Class type: AnyKinSPSheetSolverType
Eval moment: Const
Default: SPSolverModes

The SolverType defines one of four solver for the wrapping problem. Possible Values are: SPSolverFull - standard optimization solver for constraint optimization; SPSolverModes - method using modes to solve the optimization problem. This method is default and faster especially for fewer points in contact.
