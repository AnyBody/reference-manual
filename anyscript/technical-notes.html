Technical notes¶

This section is dedicated to elaborate on certain areas of the musculoskeletal modelling process inside the AnyBody Modeling System (AMS).

We have described certain topics to help the user understand and make informed decisions regarding their modeling tasks.

We use the format of technical notes to describe these topics. Each topic will contain a short introduction followed by links to related resources and a link to download the technical notes related to the topic.

Metabolic power¶

The AnyBody Modeling System have the ability to calculate the metabolic power consumed by the model during a simulation. It is denoted Pmet and it is calculated for each muscle model in the model, and the collective measure is part of the selected output of the AnyBodyStudy class.
